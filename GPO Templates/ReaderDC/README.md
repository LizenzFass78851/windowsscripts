## Adobe Reader DC 32 & 64 Bit Version GPO Template

This contains the Group Policy templates for Adobe Reader DC in the 32-bit and 64-bit versions.

Original source: https://github.com/nsacyber/Windows-Secure-Host-Baseline/tree/master/Adobe%20Reader/Group%20Policy%20Templates

- The `ReaderDC.adml` file is intended for the `en-us` language.
- The `ReaderDC.admx` file from the linked source was extended using AI-Coding Tools (GH Copilot) so that all GPOs can be set separately for 32-bit and/or 64-bit systems.

Tested with Adobe Reader DC in both 32-bit and 64-bit versions, version 26.001.21677
